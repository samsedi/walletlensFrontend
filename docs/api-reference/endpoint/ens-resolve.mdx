## Query Parameters

<ParamField query="address" type="string" required>
  The wallet address or ENS domain to resolve (e.g., `vitalik.eth`).
</ParamField>

<ParamField query="refresh" type="boolean" default="false">
  Forces a cache bypass to fetch live ENS data.
</ParamField>
